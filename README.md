# 🚀 OptimusTech: Landing Page Institucional (HTML & CSS/Flexbox)

Este projeto consiste na criação de uma Landing Page completa e responsiva para a OptimusTech, uma empresa fictícia, utilizando HTML e CSS puro.

O foco principal do desenvolvimento foi a aplicação de conceitos avançados de **Flexbox** para garantir um layout fluido e a implementação de **Media Queries** para assegurar a responsividade total em diferentes tamanhos de tela.

## 🎯 Objetivo

Criar uma Landing Page que demonstre:
1.  **Layout Profissional:** Estrutura clara e organizada para apresentar informações sobre a empresa, vagas e métricas.
2.  **Utilização de Flexbox:** Domínio na organização de elementos em diversas seções, como a navegação (`.container-navbar`), métricas (`.container-metricas`) e descrição de vagas (`.descricao-vagas-abertas`).
3.  **Design Responsivo:** Otimização para dispositivos móveis (com o uso de `@media screen and (max-width: 768px)`), garantindo uma experiência de usuário consistente.

## 📁 Estrutura do Projeto

* `index.html`: Estrutura principal da página (semântica HTML).
* `style.css`: Estilização completa, incluindo variáveis CSS (root), flexbox e media queries.
* `logoimg.png`: Imagem do logo da empresa.
* `timeimg.png`: Imagem da equipe, usada na seção de vagas.
* `Avatar.png`: Imagem de avatar para a seção de feedback.

## 🔑 Destaques Técnicos

* **Variáveis CSS (`:root`):** Uso de variáveis para cores, facilitando a manutenção e a consistência visual.
* **Flexbox em Componentes Chave:** Implementação eficiente de `display: flex` em todos os componentes principais para alinhamento e distribuição de espaço.
* **Design System Básico:** Definição de estilos para tipografia (fonte Inter), botões e caixas de informação.
* **Seções Implementadas:**
    * Navegação (`Header`)
    * Cabeçalho Principal (`Sobre nós`)
    * Métricas de Desempenho
    * Vagas Abertas (Design e Desenvolvimento de Software)
    * Feedback de Colaboradores
    * Formulário de Newsletter e Rodapé

## ⚙️ Como Visualizar

Basta abrir o arquivo `index.html` em qualquer navegador web moderno.
